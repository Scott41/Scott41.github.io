Shopper

A mobile app to help you grocery shop. This app utilizes the PhoneGap/Cordova build process to package a web app as a mobile app. Thus far, I have aimed my UI development towards the iPhone (though iPads don't render things too terribly).
I have recently finished the earliest working version of the app. I still have a long list of TODOs such as having a legitimate icon and splash screen icon/image, adding global user options, and to expand the core functionality to allow more detailed grocery lists as well as collecting data on buying habits for the user's benefit.


Libraries and tools used:
Backbone.js Marionette.js jQuery Browserify Sass PhoneGap
